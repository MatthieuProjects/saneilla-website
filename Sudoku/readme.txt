Installez les packages nécessaires en utilisant npm (ou yarn) :
npm install express socket.io

Pour exécuter le serveur, utilisez la commande suivante dans le répertoire où vous avez créé le fichier server.js :
node server.js

Ensuite, ouvrez votre navigateur et accédez à http://localhost:3000 pour l'interface utilisateur du chat.

Pour mettre le main.js donc le pong, dans une page HTML pong : 
<main>
    <canvas id="canvas" width="640" height="480"></canvas> 
</main>
<script src="js/main.js"></script>

avec html 5 qui va nous permettre de dessiner le plateau et autres éléments du jeu-->
